## 📦 Instalação Rápida

### Pré-requisitos

```bash
✓ Node.js >= 16.x
✓ MongoDB >= 5.0
✓ Git
✓ npm ou yarn
```

### Setup Local

```bash
# 1. Clone o repositório
git clone https://github.com/jjaquelinepereira1982-alt/hbc-studios-ia.git
cd hbc-studios-ia

# 2. Instale as dependências
npm install

# 3. Configure as variáveis de ambiente
cp .env.example .env

# 4. Inicie o servidor
npm run dev
```

**Acesse:** `http://localhost:3000`
